Designer Quick Start Guide

When generating your customized layout themes, create document layers matching the exact metrics below. Export files as PNG-24 with Transparency checked.

Photoshop / Illustrator

Create a document set to 1200 px width and 1800 px height. Make sure Resolution is set to 300 Pixels/Inch and Color Mode is RGB Color.

Canva Layout bounds

Use "Custom Size" when starting a layout and input 1200 x 1800 px. Design your text branding layers at the margins, keeping image box segments completely transparent.

Hardware Calibration

The 3-Photo Strip layout places image centers at exact intervals: $Y = [350, 900, 1450]$. Design boundaries accordingly to prevent covering portrait subjects.
